# SaaS Subscription Management Platform - MVP

## Project Overview

This is a SaaS platform MVP to help users track and manage their subscriptions easily. It will include features like subscription management, reminders, payment tracking, and analytics.

## Tech Stack

- Backend: Express.js
- Database: MongoDB
- Authentication: Firebase Google OAuth
- Email Notifications: SendGrid or similar

## Features (Planned)

- User signup and login with Google OAuth
- Add, edit, view, and delete subscriptions
- Dashboard for spending insights
- Email reminders before renewals
- Manual payment logging

## Getting Started

- Clone the repo
- Install dependencies
- Configure environment variables
- Run the backend server

## API Endpoints (Planned)

- Authentication routes
- Subscription CRUD routes
- Payment logging routes
- Notification preferences routes
